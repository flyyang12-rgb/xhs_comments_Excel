给同事的使用步骤：

1. 解压整个文件夹，不要只拿 xhs_chrome_extension。
2. 双击“启动小红书签名服务.bat”。
3. 黑色窗口显示 Sign server is already running 或 Starting XHS sign server 后，不要关闭窗口。
4. 打开 Chrome，进入 chrome://extensions/。
5. 打开右上角“开发者模式”。
6. 点击“加载已解压的扩展程序”，选择 xhs_chrome_extension 文件夹。
7. 在 Chrome 里登录小红书。
8. 点击插件，输入关键词和采集条数，开始采集。

如果双击 bat 提示 Node.js was not found：

说明这台电脑没有 Node 运行环境。请把 node.exe 放到当前文件夹，或者放到 node/node.exe。
