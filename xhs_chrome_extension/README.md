# 小红书关键词笔记评论采集 Chrome 插件

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`。
2. 开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录：`xhs_chrome_extension`。

## 使用

1. 双击项目根目录的 `启动小红书签名服务.bat`，保持黑色窗口不要关闭。
2. 先在当前 Chrome 中打开并登录 `https://www.xiaohongshu.com/`。
3. 点击插件图标，输入关键词，例如 `儿童雨衣`，选择采集条数 `1-20`。
4. 插件会自动读取当前 Chrome 的小红书登录态，直接请求 PC 端接口采集“最多评论”排序的笔记，并继续采集全部一级评论和二级评论。
5. 采集完成后点击“导出”，浏览器会下载：
   - `notes_raw.xlsx`
   - `comments_raw.xlsx`

## 说明

- 插件会自动读取当前 Chrome 的小红书 Cookie，并使用其中的 `a1` 生成签名。
- 签名逻辑使用本机 Node 服务执行，和 Spider_XHS 的 Node/execjs 思路一致；服务只负责生成请求头，不保存账号信息。
- 关键词搜索阶段固定使用 `comment_descending`，对应“最多评论”排序。
- 如果出现安全限制、验证码或接口失败，插件会停止当前轮采集，已采集的数据仍可导出。
- 请求间隔默认 15 秒，可在弹窗中调大以降低风控概率。
